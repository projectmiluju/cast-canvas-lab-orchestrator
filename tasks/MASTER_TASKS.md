# CastCanvas Lab — Master Task Tracker

이 파일은 **전체 레포의 모든 작업 지시 단일 소스**다.

- 모든 레포 에이전트는 작업 시작 전 이 파일을 읽는다
- 작업을 완료하면 이 파일의 상태를 업데이트한다
- 새 작업이 필요하면 이 파일에 먼저 추가한다

---

## 범례

- `[x]` 완료
- `[~]` 진행 중
- `[ ]` 미시작
- `[-]` 제외 (MVP 범위 밖)

## 에이전트별 작업 규칙

### FE 에이전트
1. 이 파일에서 `FE 독립 작업` 또는 `통합 작업` 중 `[ ]` 항목 확인
2. 구현
3. `docs/features/` 업데이트 (feature 구조 변경 시)
4. `docs/decisions/` 추가 (비자명한 결정 시)
5. `docs/til/` 추가 (세션 요약)
6. 이 파일 해당 항목 `[x]`로 업데이트

### BE 에이전트
1. 이 파일에서 `BE 독립 작업` 중 `[ ]` 항목 확인
2. 구현
3. `STATUS.md` 업데이트
4. 이 파일 해당 항목 `[x]`로 업데이트

### COLLAB 에이전트
1. 이 파일에서 `COLLAB 독립 작업` 중 `[ ]` 항목 확인
2. 구현
3. 이 파일 해당 항목 `[x]`로 업데이트
